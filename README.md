# Installation

# composer et yarn :
- composer install,
- yarn install.

# base de données:
- php bin/console d:d:c,
- php bin/console d:s:c.

# chargement des fixtures
- php bin/console d:f:l (yes).

# chargement des fichiers js et css
- yarn run dev.

# Administration

la route d'accés à la page admin est /admin (username: admin, password: password).
